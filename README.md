# taSki DevOps Assessment — CI/CD Pipeline

A CI/CD pipeline for a Node.js app using Jenkins, Docker, and AWS EKS.

---

## What This Does

Push code to GitHub → Jenkins builds, tests, and deploys to EKS automatically.

---

## Tools Used

- Jenkins
- Docker + DockerHub
- AWS EKS
- Kubernetes
- Node.js

---

## Project Structure

devops-taski-assessment/
├── app/                  # Node.js application

├── k8s/                  # Kubernetes manifests
├── scripts/              # Health check and rollback scripts
